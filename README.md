This program is designed to play the musical game "Guess my song"
Principle of operation:
1. After logging in, you need to copy the link (URL) of the desired song from the Youtube platform 
into the designated field and press the "ADD" button. The link will be saved, no one will see it and then you can 
pass the device to another player, who will have to repeat the same operation with the device.
2. Once the playlist is assembled, the "Shuffle" button is pressed to shuffle the playing sequence.
3. When the playlist is shuffled, the "Play" button is pressed and the first track in the sequence starts playing in the newly opened window. After playing the track, the newly opened window is closed and the "Play next" button is pressed and the second track in the sequence is played in this way. This is repeated until all tracks in the sequence have been played.
4. When all playlist items have been played, the message "All URLs have been played!" and the button "Reset & Play again" will appear. The "Reset & Play again" button allows you to start the playback sequence from the beginning. To create a new playlist, you need to click the "Clear URLs list" button and then you will be able to add new tracks.
